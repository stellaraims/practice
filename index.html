CSS Tables
Cheat Sheet
Tag Name Description
<table> Table Applies to the entire table
<thead> Table header Holds the <th> tag where you use colspan=""
<tr> Table row Holds table data (aka cells) from left (top) to right (right)
<td> Table data This is where the text content actually goes
<tfoot> Table footer Similar to <thead>, also contains <th>
